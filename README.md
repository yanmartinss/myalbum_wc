# ⚽ MyAlbum - World Cup Virtual Sticker Album

A full-stack web application that allows users to manage, track, and trade World Cup stickers digitally. Users can manage their collection, view duplicate stickers, and perform secure, transactional trades.

## 🚀 Features

- **Sticker Dashboard:** Track your overall progress, missing stickers, and duplicates with a clean, sporty UI.
- **Smart Collection Management:** Easily increment or decrement sticker quantities.
- **Secure Trade System:** Perform atomic trades (giving/receiving stickers) backed by Prisma transactions to ensure data consistency.
- **Fully Containerized:** Multi-container Docker setup for seamless local development with Hot Reloading (HMR).

## 🛠️ Tech Stack

**Frontend:**

- React.js

**Backend:**

- Node.js & TypeScript
- Express
- Prisma ORM
- PostgreSQL

**DevOps & Tools:**

- Docker & Docker Compose
